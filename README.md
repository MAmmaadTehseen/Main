# FYP COMPASS

A Final Year Project management system for students and advisors.

## Project Structure

```
Main/
├── Backend/          # Express.js + MongoDB API
└── frontend/         # React + Vite frontend
```

## Getting Started

### Backend Setup

```bash
cd Backend
npm install
npm run dev
```

The backend runs on **http://localhost:5000**

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

The frontend runs on **http://localhost:5173**

## Features

- User authentication (Login/Signup) with JWT
- Role-based access (Admin, Advisor, Student)
- Project management
- Task creation and submission
- File uploads
- Discussion board
- Progress tracking
- Chatbot assistant
